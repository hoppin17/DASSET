# 가격 데이터 수집 메모

프로젝트에서 가격 데이터를 수집하며 검토한 거래소 API와 라이브러리를 정리했습니다. TradingView를 이용한 수집은 유료 요금제가 필요해 이 문서에서는 다루지 않습니다.

## 바이낸스 API

바이낸스는 지원하는 코인의 종류가 다양하고 거래 데이터를 비교적 잘 제공해 가장 활용하기 수월했습니다.

## 업비트 API

업비트는 국내 주요 거래소로 API를 이용하기 어렵지 않았습니다. 다만 상장폐지된 자산의 과거 데이터를 제공하지 않아 WEMIX 가격 데이터는 확보할 수 없었습니다.

## CCXT

CCXT 라이브러리를 이용하면 여러 거래소의 가격 데이터를 비슷한 방식으로 수집할 수 있었습니다. 다만 조회할 수 있는 기간과 데이터 개수는 거래소별 정책에 따라 달랐습니다.

특히 빗썸은 당시 API에서 최근 3,000건까지만 제공해 전체 기간 데이터를 수집하는 데 한계가 있었습니다. 대체 수집 방법을 찾기 어려워 제공되는 범위 안에서 데이터를 사용했습니다.
