# Data

이 저장소에는 기업 요청과 파일 크기 문제로 원본 및 중간 데이터셋을 포함하지 않습니다.

노트북은 실행 위치와 관계없이 저장소 루트를 찾은 뒤, 아래 상대경로에서 데이터를 읽고 씁니다.

- `raw/`: API 등에서 수집한 원본 JSON·CSV
- `interim/`: 전처리 중간 산출물
- `processed/`: 분석에 바로 사용하는 정제 데이터

세 폴더는 Git에서 제외됩니다. 필요한 데이터는 로컬에서 별도로 준비하거나 데이터 수집 노트북으로 생성합니다. 분석 결과물은 저장소 루트의 `results/` 아래에 저장합니다.
